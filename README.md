# Cárcel de los Gemelos

This project implements the design and technical specifications for the "Cárcel de los Gemelos" reality show game.

## Project Structure
- `docs/`: SDD Foundation documents (Constitution, Spec, Plan).
- `scripts/`: Prototype simulation scripts.
- `src/`: Future implementation of the authoritative server (Go).

## Status: SDD Foundation Ready
The project laws, game mechanics, and technical architecture have been defined using Spec-Driven Development (SDD).
