# Cárcel de los Gemelos

This project implements the design and technical specifications for the "Cárcel de los Gemelos" reality show game.

## Project Structure
- `docs/`: SDD Foundation documents (Constitution, Spec, Plan, Tasks).
- `server/`: The authoritative game server written in Go (Clean Architecture, Event Sourcing).
- `client/`: The "Panóptico" Next.js frontend for monitoring and visualization.
- `scripts/`: Prototype simulation scripts.
- `test/`: Shadow Mode testing suite for engine validation.

## Status: Backend V2.1 Hardcore Pivot Complete
The project laws, game mechanics, and technical architecture have been defined using Spec-Driven Development (SDD). The Go backend has successfully implemented Phase 10 (New Cast Adaptation & Hardcore Mechanics) and is fully tested. Next step is expanding the Frontend / Client.
